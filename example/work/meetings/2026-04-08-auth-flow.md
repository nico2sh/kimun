## Decisions

- Support both authorization code flow (server-side apps) and PKCE (SPAs/mobile)
- Token refresh handled by client SDK, not server — reduces server complexity
- Access token TTL: 15 minutes
- Refresh token TTL: 7 days
- New endpoints: `/oauth/authorize`, `/oauth/token`, `/oauth/revoke`
