# Using Kimün

Here you can find more details of Kimün's features, the different ways to use it (a CLI or the full Terminal UI), but also how you can automate it with your favorite AI tool. Check the [skills](@/ai-skills.md) or the [MCP Server](@/ai-mcp-server.md), there you will get instructions on how to set it up and the different use cases you can use AI assist combined with Kimün.

Also you will find references on how to customize the TUI theme, and even create your own if the ones available are not good enough for you. If you are just getting started, you will probably want the TUI walkthrough first; the CLI and AI pages are reference material you can dip into as you need them.

**Start here:** [Terminal UI](@/using-kimun/tui.md)
