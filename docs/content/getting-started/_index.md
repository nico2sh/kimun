# Getting Started

Let's get started! This section walks you through installing Kimün, pointing it at a workspace, and getting your first notes indexed. If you're brand new, start with **Installation**, then jump into the **Configuration** to understand the options you should care about.

Keep in mind that you can always test the app with some sample notes. In the main repository, check the `example` directory, which contains two note directories and a sample config (`kimun_config.toml`) pointing to these so you can browse around and mess with the notes before you feel comfortable using Kimün with your own notes. Check the `README.md` file for more details on quickly setting up a sample note workspace.

**Start here:** [Installation](@/getting-started/installation.md)
