# Using Kimün

Here you'll find more details on Kimün's features, the different ways to use it (a CLI and a full Terminal UI), and how to automate it with your favorite AI tool. Check out [Skills](@/using-kimun/ai-skills.md) or the [MCP Server](@/using-kimun/ai-mcp-server.md) — both pages walk through setup and the different ways you can combine AI assistants with Kimün.

You'll also find references on how to customize the TUI theme, and even create your own if none of the built-in ones fit. If you're just getting started, the TUI walkthrough is probably what you want first; the CLI and AI pages are reference material you can dip into as you need them.

**Start here:** [Terminal UI](@/using-kimun/tui.md)
