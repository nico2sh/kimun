# Autosave Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add autosave to the TUI editor — periodic saves, save on note switch, save on quit, and an unsaved-changes indicator in the editor title.

**Architecture:** Dirty tracking lives in `TextEditorComponent` via a `last_saved_text` snapshot. A tokio interval task fires `AppMessage::Autosave` every N seconds; the editor screen handles it along with two other save triggers (open new note, quit). Quit-save is delivered via a new `AppScreen::on_exit` lifecycle hook called by `main.rs` before returning.

**Tech Stack:** Rust, Tokio (`tokio::time::interval`, `JoinHandle`), ratatui-textarea, serde (for settings field), kimun_core `NoteVault::save_note`.

- [ ] **Step 3: Add `try_save` and update `open_path` to save + spawn timer**

  Add `try_save` to the second `impl EditorScreen` block (the one with `focus_editor` etc.).

  Note on borrowing: `try_save` takes `&mut self` and accesses `self.vault` (an `Arc`), `self.path`, and `self.editor`. Because `Arc::save_note` takes `&self` (shared ref) and `VaultPath` is `Clone`, Rust's NLL allows all three subfield borrows simultaneously. No explicit cloning is needed inside `try_save`.

  ```rust
  async fn try_save(&mut self) {
      if self.editor.is_dirty() {
          let text = self.editor.get_text();
          if self.vault.save_note(&self.path, &text).await.is_ok() {
              self.editor.mark_saved(text);
          }
      }
  }
  ```

  Update `open_path` to save before loading, then spawn the timer at the end. The full updated method:

  ```rust
  pub async fn open_path(&mut self, path: VaultPath, tx: &AppTx) {
      // Save current note before switching
      self.try_save().await;

      self.path = path.clone();
      let content = self.vault.get_note_text(&self.path).await.unwrap();
      self.editor.set_text(content);

      // Only load the sidebar on first open (when it has no entries yet).
      if self.sidebar.is_empty() {
          let dir = if path.is_note() {
              path.get_parent_path().0
          } else {
              path
          };
          self.navigate_sidebar(dir, tx).await;
      }

      // Abort any existing timer and spawn a fresh one for the new note.
      if let Some(h) = self.autosave_handle.take() {
          h.abort();
      }
      let interval_secs = self.settings.autosave_interval_secs;
      let tx2 = tx.clone();
      self.autosave_handle = Some(tokio::spawn(async move {
          let mut interval = tokio::time::interval(std::time::Duration::from_secs(interval_secs));
          interval.tick().await; // skip immediate first tick
          loop {
              interval.tick().await;
              if tx2.send(AppMessage::Autosave).is_err() {
                  break;
              }
          }
      }));
  }
  ```
